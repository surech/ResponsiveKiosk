# ResponsiveKiosk
Eine einfache Möglichkeit, wie responsive Webseiten in doppelter Grösse z.B. für einen Kiosk-Computer angezeigt werden können.

Die Seite kann hier direkt angesehen werden:
http://www.surech.ch/responsiveKiosk/kiosk.xhtml

Die angezeigte URL kann per Parameter übergeben werden:
[ULR](http://www.surech.ch/responsiveKiosk/kiosk.xhtml?url=http://www.swiss.com)
